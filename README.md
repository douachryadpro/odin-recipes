# Odin Recipes

This project is part of the Foundations course from The Odin Project.  
The goal is to build a basic multi‑page recipe website using only HTML.  
The project is developed step‑by‑step through four iterations to practice structure, links, lists, images, and file organization.

---

## Iteration 1: Initial Structure
- Created the `odin-recipes` directory.
- Added an `index.html` file containing standard HTML boilerplate.
- Included an `<h1>` heading titled **"Odin Recipes"**.

---

## Iteration 2: First Recipe Page
- Created a `recipes/` directory inside the main project folder.
- Added the first recipe page (e.g., `lasagna.html`) with its own HTML boilerplate and an `<h1>` heading.
- Added a link from `index.html` to the recipe page using a relative path.
- Added a link back to the homepage (`../index.html`) inside the recipe page for easier navigation.

---

## Iteration 3: Recipe Page Content
Each recipe page includes:
- An image of the finished dish placed under the main heading.
- A **Description** section with one or two paragraphs.
- An **Ingredients** section containing an unordered list.
- A **Steps** section containing an ordered list of instructions.

---

## Iteration 4: Additional Recipes
- Added two more recipe pages with the same structure as the first one.
- Updated the `index.html` file to include links to all recipes.
- Organized the recipe links inside an unordered list for better readability.

---

## Skills Practiced
- Writing semantic and structured HTML  
- Using relative links for navigation  
- Creating multi‑page websites  
- Working with headings, lists, and images  
- Organizing files within a project  
- Using Git and GitHub for version control  
- Following step‑by‑step development instructions
